# Fuzzy Logic: Anomaly Detection

This repository contains the implementation of the Fuzzy Inference Systems (FIS) 
discussed in the paper "RegTech for Bid-Rigging: A Fuzzy-Logic Screening Approach".

## Contents
- `code/`: Implementation of Mamdani, TSK-0, and TSK-1 architectures.
- `supplementary/`: Detailed appendices, including clustering analysis and rule matrices.
- `requirements.txt`: Necessary Python libraries.

## How to use
The TSK-1 model is implemented using the `simpful` library. To reproduce the 
results, install the dependencies and run the cross-validation scripts.

*Note: This repository is anonymized for double-blind peer review.*
